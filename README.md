# Temple Pre-Marriage Registration System

A comprehensive system for temple marriage registration and village profile management.

## Features
- **User Authentication**: Secure sign-up and sign-in for devotees.
- **Marriage Registration**: Detailed multi-step registration form.
- **Admin Panel**: Management of registration records with dynamic Application Number syncing.
- **Village Profile**: Integration with village administration.
- **Payment Integration**: Razorpay integration for fee payments.
- **Certificate Generation**: Dynamic marriage certificates.

## Project Structure
- `/backend`: Spring Boot application (Java, Maven).
- `/`: Frontend HTML, CSS, and JavaScript files.

## Technical Stack
- **Backend**: Java 17, Spring Boot, Spring Data JPA, MySQL.
- **Frontend**: HTML5, Vanilla CSS3, JavaScript (ES6+), FontAwesome.
- **Database**: MySQL.
